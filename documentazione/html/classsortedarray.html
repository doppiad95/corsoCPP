Detailed Description

template<typename T, typename F>
class sortedarray< T, F >


Array di tipo T

La classe contiene due array uno con i dati non ordinati e uno con le posizioni in maniera tale che i dati se letti leggendo sequenzialmente le posizioni dal secondo array risultino ordinati
